# autosmosis

## testing

first start the tests

```
cd ./packages/autosmosis
yarn test:watch
```

hit "p" and then type a search string to scope to your test, e.g., "keplr.bank"

## next server

if you need to edit `autosmosis` and view changes in next

```sh
cd ./packages/next
yarn dev
```
